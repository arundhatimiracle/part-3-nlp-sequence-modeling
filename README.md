# NLP and Sequence Modeling Mini Project

This project focuses on Natural Language Processing (NLP) and sequence modeling techniques.

The project includes:

- Dataset understanding
- Text preprocessing
- Text vectorization using TF-IDF
- Baseline model creation
- Sequence model understanding (LSTM/RNN)
- Attention and Transformer reflection

Libraries used:
- Pandas
- NumPy
- Scikit-learn
- NLTK
- TensorFlow
- Matplotlib
